Pre-procesamiento¶

Los datos descargados contienen columnas y registros que requieren procesarse para eliminar inconsistencias. En este archivo se crea una fecha unificada llamada fecha infeccioso, que toma la primera fecha entre los datos de fechas (de notificación, de diagnóstico, de reporto) incluidos en el conjunto datos. Luego se crea un dataframe con un indice que incluye todas fechas. Con este índice se crea un dataframe con las siguientes columnas:

dias: Dias transcurridos desde la fecha inicial del conjunto de datos.
semana: Semanas trasncurridas desde la fecha inicial del conjunto de datos.
infectados: Número de casos nuevos por día.
recuperados: Número de personas recuperadas.
muertos: Número de muertos.
acumulado_infectados: Número de infectados.
acumulado_recuperados: Número de personas recuperados.
acumulado_muertos: Número de muertos totales.
activos: Total de casos activos.
fecha_infeccioso: Fecha construida a partir de la menor fecha posible entre: fecha de notificación, fecha de diagnóstico y fecha de inicio de sintomas.

Finalmente se exportan el archivo con el dataframe con una extensión .pickle.

Creacion serie de tiempo de Casos Activos¶

Los casos activos por cada día equivalen a todos los casos del dataframe en el dia mencionado, menos los recuperados, menos los muertos. Creamos un indice que incluye todas las fechas en el intervalo de análisis. Para esto detectamos la fecha inicial y final y usamos la funcion np.arange Se agrega una caracteristica al dataframe que indica un intervalo de 7 días desde la fecha del primer contagio, esta caracteristica se usa para el modelo de predicción a mediano plazo.

import datetime

#Se crea el data frame con todas las fechas en el itervalo
def CreateDates(str_date,df_covid):
  df_proc = df_covid.copy()
  #Se crea el indice por fecha
  dates = df_proc.groupby(str_date).count()
  date_init = dates.index[0]
  date_end = dates.index[-1]

  #Se crea un array con todos los días, se agrega un dia para incluir los limites
  all_days_arr = np.arange(date_init,date_end + datetime.timedelta(days=1))
  all_dates = pd.DataFrame(all_days_arr)
  all_dates['dias'] = np.arange(0,len(all_days_arr),step=1)

  #Se calculan intervalos de 7 dias que serán asignados a cada fecha. (Esta caracteristica se utiliza despues para la predicción a     mediano plazo) 
  spans_count = np.int(np.ceil(len(all_days_arr)/7))
  spans = np.repeat(np.arange(0,spans_count),7)
  all_dates['semana'] = spans[0:len(all_days_arr)]
  
  print(date_init,"date init")
  print(date_end,"date end")
  all_dates.set_index(0,inplace=True)
  return all_dates

  #Se calculan los acumulados para las columnas, Infectados, Recuperados, Muertos, Activos
def GetActivos(dfCovid):
  df_proc = dfCovid.copy()
  dfActivos = CreateDates('fecha_infeccioso',df_proc)
  dfActivos['infectados'] = df_proc.groupby(['fecha_infeccioso']).count()['id_de_caso']
  dfActivos['recuperados'] = df_proc.groupby(['fecha_recuperado']).count()['id_de_caso']
  dfActivos['muertos'] = df_proc.groupby(['fecha_de_muerte']).count()['id_de_caso']
  dfActivos.fillna(0,inplace= True)
  dfActivos['acumulado_infectados']= dfActivos['infectados'].cumsum()
  dfActivos['acumulado_recuperados'] =  dfActivos['recuperados'].cumsum()
  dfActivos['acumulado_muertos'] =  dfActivos['muertos'].cumsum()
  dfActivos['activos']= dfActivos['acumulado_infectados'] - dfActivos['acumulado_recuperados'] - dfActivos['acumulado_muertos']
  dfActivos.index.rename("fecha_infeccioso",inplace=True)

  dfSemanales = dfActivos.groupby(['semana']).sum()[['infectados','recuperados','muertos']]
  dfSemanales['acumulado_infectados']= dfSemanales['infectados'].cumsum()
  dfSemanales['acumulado_recuperados'] =  dfSemanales['recuperados'].cumsum()
  dfSemanales['acumulado_muertos'] =  dfSemanales['muertos'].cumsum()
  dfSemanales['activos']= dfSemanales['acumulado_infectados'] - dfSemanales['acumulado_recuperados'] - dfSemanales['acumulado_muertos']

  return dfActivos, dfSemanales

#Se Genera la grafica con los casos diarios y los acumulados
def GraphData(label,dfActivos,dfSemanales = None):
  fig= plt.figure(figsize=(30,5))
  ax = plt.subplot(1,4,1)
  dfActivos[['infectados','recuperados','muertos']].plot(ax=ax)
  plt.title(label)
  ax = plt.subplot(1,4,2)
  dfActivos[['activos','acumulado_infectados']].plot(ax=ax)
  plt.title(label)
  if dfSemanales is not None:
    ax = plt.subplot(1,4,3)
    dfSemanales[['infectados','recuperados','muertos']].plot(ax=ax)
    plt.title('Semanal - {}'.format(label))
    ax = plt.subplot(1,4,4)
    dfSemanales[['activos','acumulado_infectados']].plot(ax=ax)
    plt.title('Semanal - {}'.format(label))

Almacenamiento y visualización¶

Se crea un dataframe filtrado por cada ciudad a partir de dataframe general, se usa la función GetActivos(dfCovid[(dfCovid['ciudad_de_ubicaci_n'] == ciudades[i])]). Luego se almacenan los dataframe mediante las instrucciones df.to_pickle('Datos/data_{}.pickle'.format(names[i])) y dfw.to_pickle('Datos/data_weekly_{}.pickle'.format(names[i])). Adicionalmente se visualiza cada dataframe para verificar la integridad de la información, se usa la funció GraphData(ciudades[i],df,dfw).
